The Array Summation Sensation!

In each text box below, enter a 2D array, where rows are separated by
semicolons, and items in each row are separated by a white space. For example,
in case of the following two-dimensional array:

100 101 102
200 201 202
300 301 302

you should enter:
100 101 102;200 201 201;300 301 302
